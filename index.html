Redesigned the Berkeley's DeCal website with Ruby on Rails. On this project, I worked on designing the schema of the database, creating different interfaces for faculty and student, and added logic for how classes should be displayed and interacted with.

A personal script that I made using Selenium and Python which can play the online game of Mousehunt. This script sounds the horn every 15 minutes and was able to warn the user via email whenever a captcha would come up. I'm currently working on automating that process with scikit-image to resolve the Mousehunt captcha.
